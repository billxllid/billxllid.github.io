# 正则表达式Tips

## 大小写

- **不区分大小写：** 下面是两种写法：

``` python
>>> re.findall('(?i:p)ython', data)
['python', 'Python']

>>> re.findall('(?i)python')
['python', 'Python']
```

- **区分大小写：** 默认是区分的，强制区分：

``` python
>>> re.findall('(?-i:p)ython')
['python']
```

## 数量

- `*`：0或更多次
- `+`：1或更多次
- `?`：0或1次

## 贪婪/非贪婪

默认是贪婪模式，加上`?`后变为非贪婪模式：

- `*?`、`+?`、`??`均为非贪婪模式

``` python
>>> data = '<a> b <c>'
>>> re.findall('\<.*\>', data)
['<a> b <c>']
>>> re.findall('\<.*?\>', data)
['<a>', '<c>']
```

- `{m,n}?`同样转化为非贪婪模式

``` python
>>> data = 'aaaaaa'
>>> re.findall('a{3,5}', data)
['aaaaa']
>>> re.findall('a{3,5}?', data)
['aaa', 'aaa']
```
